**AniBT** 是一個面向動漫的公共種子追蹤器，專注服務動漫與字幕組。我們希望給字幕組一個可靠的釋出平臺，也讓使用者和工具更容易自動追到想看的內容。

這份 Wiki 按用途分兩條主線：**字幕組釋出資源** 和 **使用者/開發者消費資料**。挑你屬於的那一邊讀起。

---

## 我是字幕組，想釋出資源

完整流程：申請入駐 → 配站點同步 → 制種用我們的 tracker → 用 API 釋出。

  <Card
