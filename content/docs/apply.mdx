---
title: 申请入驻
description: 字幕组以邮件方式提交入驻申请，提供身份信息后由站务人工审核。
---

AniBT 不开放自助注册发布权限。**字幕组**入驻请按下面流程发邮件申请，站务收到后人工审核。审核通过后我们会给你开通发布账号、配置字幕组信息，之后即可使用 [发布 API](/docs/open-api/releases-publish) 推送资源、并通过 [站点同步](/docs/site-sync) 把发布同步到其它 BT 站。

## 申请要求

<Steps>
  <Step>
    ### 用注册邮箱发送
    请使用您在 AniBT 注册的邮箱发送申请邮件。我们会用这个邮箱来核对你确实是该账号本人，避免冒名申请。

    还没注册？先在站点 [anibt.net](https://anibt.net) 注册一个普通账号，再用同一个邮箱来发申请。
  </Step>
  <Step>
    ### 字幕组基本信息
    邮件正文请提供：

    - **组名**：完整的字幕组名称。
    - **slug**：字幕组的英文标识符，作为唯一 ID 使用，例如 `sakura-sub`。请尽量简短、稳定，开通后不易更改（它会出现在 [`/rss/group/<slug>.xml`](/docs/open-api/other-endpoints) 等公开 URL 里）。
    - **简要介绍**：一两段说明你们做什么类型的作品、活跃多久、目前的发布节奏。
    - **联系方式**：Telegram、官网、Discord、Email 群组等任何可以反向联系到字幕组的渠道。
  </Step>
  <Step>
    ### 身份验证信息
    提供任意一项可证明身份的材料即可，**不需要全部**：

    - 字幕组在 nyaa / 末日站 / Mikan / ACG.rip 等站点已使用的发布账号或邮箱（我们会比对）。
    - 字幕组官方网站 / Telegram 频道 / Twitter 等公开渠道的管理员证明（例如临时在频道置顶一条声明）。
    - 已在其它字幕组成员之间被广泛认可的身份证据。

    目的是确认你确实是这个字幕组的成员或负责人，而不是冒用他人名义注册 slug。
  </Step>
</Steps>

## 发送至

<Callout type="info">
  邮件发送至 **[support@anibt.net](mailto:support@anibt.net)**，主题建议写 `字幕组入驻申请 - <组名>`。我们会尽快回复，通常在几个工作日以内。
</Callout>

如果一周内没收到任何回复（含垃圾箱），可以再发一次或通过 [其他联系方式](/docs/contact) 提醒我们。

## 审核通过后

我们会回邮件确认开通，并在你的账号上：

- 绑定字幕组身份与 slug。
- 开通 [发布 API](/docs/open-api/releases-publish) 的访问权限（生成 API Key）。
- 你的字幕组会在站点列表与 [字幕组 RSS](/docs/open-api/other-endpoints) 中可见。
