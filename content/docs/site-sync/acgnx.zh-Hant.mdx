---
title: 末日站同步
description: 在 AniBT 中填寫末日站同步所需 UID 和 API Token。
---

末日站（ACGNX）需要 **UID** 與 **API Token** 兩個值，都能在帳戶 API 設定頁找到。

<Callout type="warn">
  拖曳複製貼上後，記得檢查 UID 和 API Token **前後是否有多餘空格**。
</Callout>

## 操作步驟

<Steps>
  <Step>
    ### 進入帳戶 API 設定

    造訪 [https://share.acgnx.se](https://share.acgnx.se)，點選左上角 **發佈資源**，進入頁面後在側邊欄 **帳戶控制中心** 下點選 **帳戶API設置**。
  </Step>

  <Step>
    ### 複製 UID 與 API Token

    複製 **UID** 和 **API Token**，回到 AniBT 對應欄位填寫並儲存。

    ![末日站帳戶 API 設定](/images/site-sync/acgnx-api-settings.webp)
  </Step>
</Steps>
