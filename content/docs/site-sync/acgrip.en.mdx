---
title: ACG.rip Sync
description: Fill in the API Token required by AniBT's ACG.rip sync.
---

Open [https://acg.rip](https://acg.rip), click the blue avatar icon in the top-right navigation bar, then click **Dashboard**.

Find **Personal Settings** on the right. The second item below it is **API Token**.

When copying the API Token, do not include the `tpx://acg.rip/` prefix. Only copy the token after it, for example `7723-pxxxxxxxxxxxxxxp`.
