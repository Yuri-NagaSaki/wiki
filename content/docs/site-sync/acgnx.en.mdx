---
title: ACGNX Sync
description: Fill in the UID and API Token required by AniBT's ACGNX sync.
---

ACGNX requires two values — **UID** and **API Token** — both available in the account API settings page.

<Callout type="warn">
  After dragging to copy and paste, double-check there are **no extra spaces** before or after the UID and API Token.
</Callout>

## Steps

<Steps>
  <Step>
    ### Open the Account API Settings

    Visit [https://share.acgnx.se](https://share.acgnx.se), click **發佈資源** in the top-left, then click **帳戶API設置** under **帳戶控制中心** in the sidebar.
  </Step>

  <Step>
    ### Copy UID and API Token

    Copy **UID** and **API Token** into the matching fields in AniBT and save.

    ![ACGNX account API settings](/images/site-sync/acgnx-api-settings.webp)
  </Step>
</Steps>
