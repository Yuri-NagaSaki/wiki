---
title: ACG.rip 同步
description: 在 AniBT 中填寫 ACG.rip 同步所需 API Token。
---

ACG.rip 的同步資訊只需要一項 **API Token**，從儀錶板的個人設定中複製即可。

<Callout type="warn">
  複製 API Token 時**不要帶字首** `tpx://acg.rip/`，只複製後面那一串。
</Callout>

## 操作步驟

<Steps>
  <Step>
    ### 開啟儀錶板

    造訪 [https://acg.rip](https://acg.rip)，點選頂欄右上角藍色背景的**人頭圖示**，選擇 **儀錶板**。
  </Step>

  <Step>
    ### 找到 API Token

    在右側 **個人設定** 區塊，下方第二個就是 **API Token**。

    ![ACG.rip API Token](/images/site-sync/acgrip-api-token.webp)
  </Step>

  <Step>
    ### 複製並貼上到 AniBT

    範例：

    ```txt title="API Token 格式"
    7723-pxxxxxxxxxxxxxxp
    ```

    貼上到 AniBT 的 ACG.rip 同步設定中，儲存即可。
  </Step>
</Steps>
