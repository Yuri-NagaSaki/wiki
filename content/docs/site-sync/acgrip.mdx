---
title: ACG.rip 同步填写
description: 在 AniBT 中填写 ACG.rip 同步所需 API Token。
---

ACG.rip 的同步信息只需要一项 **API Token**，从仪表盘的个人设定中复制即可。

<Callout type="warn">
  复制 API Token 时**不要带前缀** `tpx://acg.rip/`，只复制后面那一串。
</Callout>

## 操作步骤

<Steps>
  <Step>
    ### 打开仪表盘

    访问 [https://acg.rip](https://acg.rip)，点击顶栏右上角蓝色背景的**人头图标**，选择 **仪表盘**。
  </Step>

  <Step>
    ### 找到 API Token

    在右侧 **个人设定** 区块，下方第二个就是 **API Token**。

    ![ACG.rip API Token](/images/site-sync/acgrip-api-token.webp)
  </Step>

  <Step>
    ### 复制并粘贴到 AniBT

    示例：

    ```txt title="API Token 格式"
    7723-pxxxxxxxxxxxxxxp
    ```

    粘贴到 AniBT 的 ACG.rip 同步设置中，保存即可。
  </Step>
</Steps>
