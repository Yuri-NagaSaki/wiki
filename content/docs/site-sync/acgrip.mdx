---
title: ACG.rip 同步填写
description: 在 AniBT 中填写 ACG.rip 同步所需 API Token。
---

打开网站 [https://acg.rip](https://acg.rip)，点击顶栏右上角蓝色背景的人头图标，然后点击 **仪表盘**。

在右侧找到 **个人设定**，下方第二个就是 **API Token**。

复制 API Token 时，不需要复制前面的 `tpx://acg.rip/`，只需要复制后面这一串，例如 `7723-pxxxxxxxxxxxxxxp`。
