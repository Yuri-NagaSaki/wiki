title: Mikan Sync
description: Fill in the API Token, publish group ID, and subgroup ID required by AniBT's Mikan sync.
---

Mikan sync needs only the **API Token** — AniBT auto-queries the publish group and release group IDs from it.

<Callout type="info">
  Usually the **API Token alone** is enough; AniBT will fetch the IDs automatically. Manual lookup steps are at the bottom of this page for reference.
</Callout>

## Steps

<Steps>
  <Step>
    ### Find your API Token

    On [https://mikanani.me](https://mikanani.me), sign in and click **Account Settings** at the top-right. Go to **Personal Center** → **Edit Profile** → find **API Token** under **Basic Profile**.

    <Callout type="info">
      If no token is visible, click **Generate** on the right.
    </Callout>
  </Step>

  <Step>
    ### Save the Token in AniBT

    Copy the API Token, paste it into AniBT's Mikan sync settings, and click **Save**.

    ![Mikan API Token and ID query](/images/site-sync/mikan-api-token.webp)
  </Step>

  <Step>
    ### Auto-query the IDs

    Click **【Query the ID of "XXXXXX" from Mikan】** — AniBT will fetch your **publish group ID** and **release group ID** automatically.
  </Step>
</Steps>

## Manual ID Lookup (optional)

Use these steps only when auto-query fails.

<Tabs items={['Publish Group ID', 'Release Group ID']}>
  <Tab value="Publish Group ID">
    The publish group ID is the trailing number in the publish group page URL.

    ```txt title="Publish group page URL"
    https://mikanani.me/Home/PublishGroup/359
    ```

    Here, `359` is the publish group ID. Click the **subtitle group** in any list or anime description — the page it opens is the publish group page.

    ![Mikan publish group ID page](/images/site-sync/mikan-publish-group-id.webp)
  </Tab>
  <Tab value="Release Group ID">
    Using LoliHouse as an example: open any anime list released under the LoliHouse identity, then click the **RSS icon** next to the subtitle group name. You'll get something like:

    ```txt title="RSS link"
    https://mikanani.me/RSS/Bangumi?bangumiId=3953&subgroupid=370
    ```

    In `subgroupid=370`, the `370` is the release group ID.

    ![Release group ID in Mikan RSS](/images/site-sync/mikan-rss-subgroup-id.webp)
  </Tab>
</Tabs>
