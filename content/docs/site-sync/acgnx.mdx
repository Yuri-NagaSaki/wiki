---
title: 末日站同步
description: 在 AniBT 中填写末日站同步所需 UID 和 API Token。
---

末日站（ACGNX）需要 **UID** 与 **API Token** 两个值，都能在账户 API 设置页找到。

<Callout type="warn">
  拖拽复制粘贴后，记得检查 UID 和 API Token **前后是否有多余空格**。
</Callout>

## 操作步骤

<Steps>
  <Step>
    ### 进入账户 API 设置

    访问 [https://share.acgnx.se](https://share.acgnx.se)，点击左上角 **發佈資源**，进入页面后在侧边栏 **帳戶控制中心** 下点击 **帳戶API設置**。
  </Step>

  <Step>
    ### 复制 UID 与 API Token

    复制 **UID** 和 **API Token**，回到 AniBT 对应字段填写并保存。

    ![末日站账户 API 设置](/images/site-sync/acgnx-api-settings.webp)
  </Step>
</Steps>
