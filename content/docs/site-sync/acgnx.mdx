---
title: 末日站同步填写
description: 在 AniBT 中填写末日站同步所需 UID 和 API Token。
---

打开网站 [https://share.acgnx.se](https://share.acgnx.se)，点击左上角的 **發佈資源**。

进入页面后，在侧边栏 **帳戶控制中心** 下方点击 **帳戶API設置**。

复制 **UID** 和 **API Token**，回到 AniBT 填写并保存。

> 拖拽复制粘贴后，记得检查 UID 和 API Token 前后是否有多余空格。
