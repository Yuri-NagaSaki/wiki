---
title: Nyaa 同步填写
description: 在 AniBT 中填写 Nyaa 同步所需 session Cookie。
---

Nyaa 通过浏览器 Cookie 鉴权，从 DevTools 中复制 `session` 字段即可。

<Callout type="warn">
  Nyaa Cookie 有效期**仅一个月**，需要定期更新。粘贴前请手动加上前缀 `session=`。
</Callout>

## 操作步骤

<Steps>
  <Step>
    ### 打开 DevTools

    在 Nyaa 已登录状态下按 <kbd>F12</kbd> 打开 **DevTools**，切换到顶栏的 **Application（应用）**。
  </Step>

  <Step>
    ### 定位 session Cookie

    在侧栏 **Storage（存储）** → **Cookie** 下展开 `https://nyaa.si`，从名称列中点击 **session**，下方会显示 Cookie Value。
  </Step>

  <Step>
    ### 复制 Cookie 值

    复制 Cookie Value 这一长串内容：

    ```txt title="DevTools 中复制到的内容"
    eyJfcGVybWFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxKFCv50xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxNTQ4OTl9.axxxxg.jC7nBl0pLs08BP-gXnktlEbIxKc
    ```

    ![Nyaa session Cookie](/images/site-sync/nyaa-session-cookie.webp)
  </Step>

  <Step>
    ### 加上 `session=` 前缀后填入

    在 AniBT 中粘贴时**手动补上前缀**：

    ```txt title="最终填入 AniBT 的内容"
    session=eyJfcGVybWFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxKFCv50xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxNTQ4OTl9.axxxxg.jC7nBl0pLs08BP-gXnktlEbIxKc
    ```
  </Step>
</Steps>
