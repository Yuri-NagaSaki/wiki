title: Mikan（蜜柑计划）同步

---

蜜柑计划同步只需要 **API Token**，AniBT 会用它自动查询字幕组 ID 与发布组 ID。

<Callout type="info">
  通常**只需 API Token** 即可——AniBT 会自动查询 ID。本页末尾给出手动获取 ID 的方法，仅供需要时参考。
</Callout>

## 操作步骤

<Steps>
  <Step>
    ### 找到 API Token

    在 [https://mikanani.me](https://mikanani.me) 登录后，点击右上角 **账号设置** → **个人中心** → **编辑资料**，在 **基本资料** 下找到 **API Token**。

    <Callout type="info">
      如果没有看到 API Token，点击右边的 **生成**。
    </Callout>
  </Step>

  <Step>
    ### 在 AniBT 中保存 Token

    复制 API Token，粘贴到 AniBT 的蜜柑计划同步设置里，点击 **保存**。

    ![蜜柑计划 API Token 与 ID 查询](/images/site-sync/mikan-api-token.webp)
  </Step>

  <Step>
    ### 自动查询 ID

    点击 **【从蜜柑查询「XXXXXX」的 ID】** 按钮，AniBT 会自动获取你的 **字幕组 ID** 与 **发布组 ID**。
  </Step>
</Steps>

## 手动获取 ID（可选）

仅当自动查询无法识别时使用。

<Tabs items={['字幕组 ID', '发布组 ID']}>
  <Tab value="字幕组 ID">
    字幕组 ID 是发布组页面 URL 末尾的数字。

    ```txt title="字幕组页面 URL"
    https://mikanani.me/Home/PublishGroup/359
    ```

    其中 `359` 即字幕组 ID。在列表或作品简介中点击 **字幕组**，跳转后的页面就是字幕组页面。

    ![蜜柑计划字幕组 ID 页面](/images/site-sync/mikan-publish-group-id.webp)
  </Tab>
  <Tab value="发布组 ID">
    以 LoliHouse 为例，进入任意以 LoliHouse 身份发布的动画列表，点击字幕组名字右侧的 **RSS 图标**，得到形如：

    ```txt title="RSS 链接"
    https://mikanani.me/RSS/Bangumi?bangumiId=3953&subgroupid=370
    ```

    `subgroupid=370` 中的 `370` 就是发布组 ID。

    ![蜜柑计划 RSS 中的发布组 ID](/images/site-sync/mikan-rss-subgroup-id.webp)
  </Tab>
</Tabs>
