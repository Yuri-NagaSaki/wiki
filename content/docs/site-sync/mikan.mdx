打开网站 [https://mikanani.me](https://mikanani.me)，登录后点击右上角的 **账号设置**。

在 **个人中心** → **编辑资料** 的 **基本资料** 下方找到 **API Token**。

> 如果没有看到 API Token，就点击右边的 **生成**。

复制这一串 **API Token**，粘贴到 AniBT 的蜜柑计划同步设置里，并点击 **保存**。

随后点击 **【从蜜柑查询「XXXXXX」的 ID】** 按钮，AniBT 会自动获取你的 **字幕组 ID** 和 **发布组 ID**。

![蜜柑计划 API Token 与 ID 查询](/images/site-sync/mikan-api-token.webp)

## 获取字幕组 ID 和发布组 ID

通常直接使用上面的自动查询按钮即可。下面是手动获取方法。

### 字幕组 ID

字幕组 ID 就是发布组页面 URL 最后的数字。

例如：

```txt
https://mikanani.me/Home/PublishGroup/359
```

这里的 `359` 就是字幕组 ID。

你可以在列表或作品简介里点击 **字幕组**，跳转后的页面就是字幕组页面。

![蜜柑计划字幕组 ID 页面](/images/site-sync/mikan-publish-group-id.webp)

### 发布组 ID

以 LoliHouse 为例，随便点进一个在 LoliHouse 身份之下发布的动画列表。

随后点击字幕组名字右边的 **RSS 图标**。

得到的 RSS 链接类似：

```txt
https://mikanani.me/RSS/Bangumi?bangumiId=3953&subgroupid=370
```

RSS 链接最后面的 `subgroupid=370` 中，`370` 就是发布组 ID。

![蜜柑计划 RSS 中的发布组 ID](/images/site-sync/mikan-rss-subgroup-id.webp)
