title: Site Sync
description: Configure AniBT release sync for Bangumi Moe, Mikan, DMHY, ACGNX, ACG.rip, and Nyaa.
---

Site sync pushes a release you publish on AniBT to the external sites you have configured, reducing duplicate uploads and manual upkeep.

<Callout type="info">
  Complete [Apply for admission](/docs/apply) first, then open your subtitle group's sync settings in AniBT. Each external site stores its own credentials.
</Callout>

## Setup Flow

<Steps>
  <Step>
    ### Choose the sites to sync

    Open the sync settings in the AniBT subtitle group dashboard and enable the external sites you actually maintain.
  </Step>

  <Step>
    ### Prepare the matching credentials

    Use the cheat sheet below to open the relevant guide and copy the required Cookie, API Token, UID, username, or password fields.
  </Step>

  <Step>
    ### Save and test the connection

    Save the configuration in AniBT, then test the connection before publishing real releases.
  </Step>
</Steps>
