---
title: RSS 主入口
description: GET /rss/anime.xml —— 支持 bgmId、字幕组 slug 与多 Tag 任意组合的 RSS 主入口，兼容 Mikan 链接。
---

`/rss/anime.xml` 是最灵活的订阅端点：用 `bgmId` + `groupSlug` + 多个 Tag（`resolution` / `language` / `format` / `subtitle` / `episode` / `episodeKey`）任意组合订阅一个番剧 / 字幕组 / 全站发布。

<Callout type="warn">
  必须给 `bgmId` 或 `groupSlug` **至少之一**。两者都不传会被拒绝。
</Callout>

## URL

```txt title="URL"
GET https://anibt.net/rss/anime.xml?<query>
```

## 重复参数两种写法等价

```txt title="标准重复 query（推荐）"
?resolution=1080p&resolution=720p
```

```txt title="逗号分隔（也支持）"
?resolution=1080p,720p
```

## 后端选择最优索引

后端根据传入参数自动选索引：

| 提供                    | 使用索引                |
| ----------------------- | ----------------------- |
| `bgmId` + `groupSlug`   | `groupId_animeId`       |
| 仅 `bgmId`              | `animeId`               |
| 仅 `groupSlug`          | `groupId_publishedAt`   |

随后在内存里按 Tag 过滤（**最多扫描 4000 条候选**），按 `publishedAt` 倒序截断 `limit`。SSR 层叠加 **300 秒缓存**（按规范化后的 query 串当 key），同一查询在缓存窗口内只命中一次后端。
