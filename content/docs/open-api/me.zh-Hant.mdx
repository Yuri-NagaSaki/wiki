title: 字幕組自身資訊

---

`/api/subtitle-groups/me` 是**鑑權讀取**端點。帶上字幕組 API Key，返回該字幕組的基本資訊（名稱、slug、簡介、網站）和統計資料（釋出數、覆蓋番劇數、成員數）。

適合用在：CI 啟動時列印當前用哪把 Key 跑、控制檯 dashboard 展示字幕組當前狀態、自動化指令碼里做"是不是登在正確字幕組"的最小校驗。

## 請求

```http title="HTTP"
GET https://site.anibt.net/api/subtitle-groups/me
Authorization: Bearer <YOUR_API_KEY>
```

```bash title="curl 示例"
curl -H "Authorization: Bearer <YOUR_API_KEY>" \
  https://site.anibt.net/api/subtitle-groups/me
```

## 與公共目錄的關係

<Callout type="info">
  與 [`/api/subtitle-groups`](/docs/open-api/subtitle-groups) 的核心差別：
  - `/api/subtitle-groups`：**公開**，返回**所有**字幕組的精簡資訊，不需要鑑權。
  - `/api/subtitle-groups/me`：**鑑權**，只返回**呼叫方**這一組的詳細資訊，附帶 `stats` 與內部 `_id`、`description`、`status`。
</Callout>

## 錯誤響應

| HTTP | 原因                            | 處理                            |
| ---- | ------------------------------- | ------------------------------- |
| 401  | `Authorization` 頭缺失或格式錯誤 | `Bearer ` 字首必填，注意空格      |
| 403  | API Key 無效 / 已撤銷           | 在後臺重新生成                  |
