---
title: 其他類 RSS
description: GET /rss/other/<category>.xml —— 漫畫、音樂、RAW、舞台劇各自獨立的 RSS，不混入主 RSS。
---

`/rss/other/<category>.xml` 輸出其他類資源的獨立 RSS。目前支援 `manga`、`music`、`raw`、`stage` 四個分類，對應漫畫、音樂、RAW、舞台劇。

<Callout type="warn">
  其他類資源**不會**混入 `/rss/magnets.xml`、`/rss/anime.xml` 或番劇詳情頁 RSS。需要訂閱這些資源時，請直接使用本頁的分類 RSS。
</Callout>

## URL

```txt title="分類 RSS"
https://anibt.net/rss/other/manga.xml
https://anibt.net/rss/other/music.xml
https://anibt.net/rss/other/raw.xml
https://anibt.net/rss/other/stage.xml
```

## 示例

<Steps>
  <Step>
    ### 訂閱漫畫最新釋出
    ```txt title="漫畫"

    ```
  </Step>
  <Step>
    ### 搜尋音樂分類

    ```txt title="RAW + 檔案大小"

    ```
  </Step>

## 快取與索引

- RSS 回應支援 `ETag` / `If-None-Match`，未變更時返回 `304 Not Modified`。
- 共享快取標頭為 `s-maxage=120`，並啟用長時間 `stale-while-revalidate` / `stale-if-error`。
- 這些 RSS 回應會帶 `noindex, follow` 搜尋引擎策略，避免分類 feed 被搜尋引擎索引。
