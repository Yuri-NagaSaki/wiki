`/api/bgm/search` 是**公開**搜尋端點，無需鑑權。用於把"使用者輸入的番劇名"對映成穩定的 Bangumi.tv subject id（`bgmId`），後續就能餵給 [`/rss/anime.xml`](/docs/open-api/rss-anime) 或 [`/api/releases/publish`](/docs/open-api/releases-publish)。

## 請求

```txt title="URL"
GET https://site.anibt.net/api/bgm/search?q=<keyword>&limit=<n>
```

支援中文、英文、日文、羅馬音任意關鍵詞。

```bash title="curl 示例"
curl -s -G 'https://site.anibt.net/api/bgm/search' \
  --data-urlencode 'q=葬送的芙莉蓮' -d 'limit=3'
```

## 實戰：搜尋 → 拼 RSS

```bash title="一行打出 bgmId 給 RSS 用"
BGM=$(curl -s -G 'https://site.anibt.net/api/bgm/search' \
  --data-urlencode 'q=葬送的芙莉蓮' -d 'limit=1' \
  | jq -r '.data[0].bgmId')

echo "https://anibt.net/rss/anime.xml?bgmId=$BGM&resolution=1080p&language=CHS"
```

## 注意事項

<Callout type="info">
  搜尋結果按 Bangumi 內部相關度排序，**不**保證第一條就是最熱門版本。如果番名容易撞車（例如有多季 / 同名作品），建議把 `limit` 設大一點，再人工挑選。
</Callout>

<Callout type="warn">
  這是**外部資料透傳**介面，結果質量直接受 Bangumi 上游影響。生產程式碼裡建議把 `bgmId` 快取到本地，不要每次都查。
</Callout>
