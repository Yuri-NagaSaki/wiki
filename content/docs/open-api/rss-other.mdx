---
title: 其他类 RSS
description: GET /rss/other/<category>.xml —— 漫画、音乐、RAW、舞台剧各自独立的 RSS，不混入主 RSS。
---

`/rss/other/<category>.xml` 输出其他类资源的独立 RSS。当前支持 `manga`、`music`、`raw`、`stage` 四个分类，对应漫画、音乐、RAW、舞台剧。

<Callout type="warn">
  其他类资源**不会**混入 `/rss/magnets.xml`、`/rss/anime.xml` 或番剧详情页 RSS。需要订阅这些资源时，请直接使用本页的分类 RSS。
</Callout>

## URL

```txt title="分类 RSS"
https://anibt.net/rss/other/manga.xml
https://anibt.net/rss/other/music.xml
https://anibt.net/rss/other/raw.xml
https://anibt.net/rss/other/stage.xml
```

## 示例

<Steps>
  <Step>
    ### 订阅漫画最新发布
    ```txt title="漫画"
    https://anibt.net/rss/other/manga.xml
    ```
  </Step>
  <Step>
    ### 搜索音乐分类
    ```txt title="音乐 + 关键词"
    https://anibt.net/rss/other/music.xml?q=FLAC&limit=50
    ```
  </Step>
  <Step>
    ### 按文件大小排序 RAW
    ```txt title="RAW + 文件大小"
    https://anibt.net/rss/other/raw.xml?sortField=fileSize&sortOrder=desc
    ```
  </Step>
</Steps>

## 缓存与索引

- RSS 响应支持 `ETag` / `If-None-Match`，未变更时返回 `304 Not Modified`。
- 共享缓存头为 `s-maxage=120`，并启用长时间 `stale-while-revalidate` / `stale-if-error`。
- 这些 RSS 响应会带 `noindex, follow` 搜索引擎策略，避免分类 feed 被搜索引擎索引。
