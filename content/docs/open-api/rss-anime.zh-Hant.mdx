title: RSS 主入口
description: GET /rss/anime.xml —— 支援 bgmId、字幕組 slug 與多 Tag 任意組合的 RSS 主入口，相容 Mikan 連結。
---

`/rss/anime.xml` 是最靈活的訂閱端點：用 `bgmId` + `groupSlug` + 多個 Tag（`resolution` / `language` / `format` / `subtitle` / `episode` / `episodeKey`）任意組合訂閱一個番劇 / 字幕組 / 全站釋出。

<Callout type="warn">
  必須給 `bgmId` 或 `groupSlug` **至少之一**。兩者都不傳會被拒絕。
</Callout>

## URL

```txt title="URL"
GET https://anibt.net/rss/anime.xml?<query>
```

## 重複引數兩種寫法等價

```txt title="標準重複 query（推薦）"

```

```txt title="逗號分隔（也支援）"
?resolution=1080p,720p
```

## 後端選擇最優索引

後端根據傳入引數自動選索引：

| 提供                    | 使用索引                |
| ----------------------- | ----------------------- |
| `bgmId` + `groupSlug`   | `groupId_animeId`       |
| 僅 `bgmId`              | `animeId`               |
| 僅 `groupSlug`          | `groupId_publishedAt`   |

隨後在記憶體裡按 Tag 過濾（**最多掃描 4000 條候選**），按 `publishedAt` 倒序截斷 `limit`。SSR 層疊加 **300 秒快取**（按規範化後的 query 串當 key），同一查詢在快取視窗內只命中一次後端。
