## 字幕组短链 `/rss/group/<slug>.xml`

等价于 `/rss/anime.xml?groupSlug=<slug>` 但 URL 更短，便于贴 README / 论坛。

<Callout type="warn">
  路径必须以 `.xml` 结尾。支持 `?limit=`，**不支持** Tag 过滤；需要过滤请走 [RSS 主入口](/docs/open-api/rss-anime)。
</Callout>

```txt title="示例"
https://anibt.net/rss/group/kirara-fantasia.xml
https://anibt.net/rss/group/pre-s.xml
https://anibt.net/rss/group/kitauji-sub.xml
https://anibt.net/rss/group/haruhana.xml
```

## 全站磁力 `/rss/magnets.xml`

适合"看看全站最近发了什么"。

```txt title="基本用法"
https://anibt.net/rss/magnets.xml
https://anibt.net/rss/magnets.xml?limit=200
https://anibt.net/rss/magnets.xml?q=葬送的芙莉莲
https://anibt.net/rss/magnets.xml?resolution=1080p&language=CHS
```

<Callout type="info">
  简单查询缓存 **120 秒**；带 `q` + 多 filter 的复杂查询缓存 **60 秒**。两者都叠加 `stale-while-revalidate=600s`，CDN 在窗口过期后仍立即吐旧版给用户、后台异步刷新。详见 [HTTP 缓存与 304](/docs/open-api/caching)。
</Callout>

## 季度番剧 `/api/seasons/anime`

第三方追番站 / 看板 / 编辑器辅助使用。**返回当季已上架番剧 + 可选季度列表。**

### 请求

```txt title="多种调用方式"
GET https://site.anibt.net/api/seasons/anime
GET https://site.anibt.net/api/seasons/anime?season=2026春
GET https://site.anibt.net/api/seasons/anime?season=2026-SPRING   # 也接受英文形式
GET https://site.anibt.net/api/seasons/anime?bgmId=543360         # 精确返回单个番剧
```

### Query

<TypeTable
  type={{
    'season': {
      description: '形如 2026春 / 2026夏 / 2026秋 / 2026冬，也接受 2026-SPRING 等 ISO 形式。',
      type: 'string',
      default: '当前季',
    },
    'bgmId': {
      description: '精确查询单个番剧（按 BGM subject id）。优先级高于 season：传入后忽略 season 过滤，仅返回该番剧；未匹配返回 total: 0 且 byWeekday: []。requestedSeason 自动回填为该番剧所属季度。',
      type: 'integer',
    },
  }}
/>

### 缓存与限流

| 项目             | 值                                                                                       |
| ---------------- | ---------------------------------------------------------------------------------------- |
| 速率限制         | **30 次 / 60 秒 / IP**（与 `/api/anime/groups` 共用窗口）                                |
| `Cache-Control`  | `max-age=600, s-maxage=600, stale-while-revalidate=86400`（**10 分钟新鲜 + 24h SWR**） |
| 条件请求         | `If-None-Match`（ETag 形如 `W/"sa-{year}-{SEASON}[-b{bgmId}]-{count}-{lastMs}-{urlHash}"`）、`If-Modified-Since` |

```bash title="拉取与 304"
curl 'https://site.anibt.net/api/seasons/anime'
curl 'https://site.anibt.net/api/seasons/anime?season=2026春'
curl 'https://site.anibt.net/api/seasons/anime?bgmId=543360'

ETAG=$(curl -sSI 'https://site.anibt.net/api/seasons/anime' | grep -i ^etag: | awk '{print $2}' | tr -d '\r')
curl -i 'https://site.anibt.net/api/seasons/anime' -H "If-None-Match: $ETAG"
# → HTTP/2 304
```

## 单番剧字幕组 `/api/anime/groups`

> 「这部番都谁在做？」一个 `bgmId` 即可拿到全部正在发布的字幕组、最近更新时间，以及**最近 30 条发布**。

**专为给 `/rss/anime.xml` 拼 `groupSlug` 提供候选**，也方便客户端直接做"番剧详情页 / 资源列表"。

### 请求

```txt
GET https://site.anibt.net/api/anime/groups?bgmId=543360
```

<TypeTable
  type={{
    'bgmId': {
      description: 'Bangumi ID（正整数），例如 543360。',
      type: 'integer',
    },
  }}
/>

### 响应（节选）

```jsonc title="200 OK"
{
  "ok": true,
  "ts": 1776841967132,
  "data": {
    "bgmId": 543360,
    "animeId": "nh73r0ernee7s0afv9s82f763983gcz7",
    "title": { /* primary / chinese / romaji / english / japanese */ },
    "groups": [
      {
        "groupId": "org_aagmgM9i0ZyR",
        "slug": "kitauji-sub",
        "name": "北宇治字幕组",
        "status": "ACTIVE",
        "lastUpdatedAt": 1776624843260,
        "items": [
          {
            "releaseId": "rel_fItEHE7r1gR1",
            "title": "[北宇治字幕组] 上伊那牡丹，酒醉身姿似百合 [02][WebRip][HEVC_AAC][繁日內嵌]",
            "magnet": "magnet:?xt=urn:btih:282e2b…",
            "episodeKey": "2",
            "resolution": "1080p",
            "language": ["CHT", "JP"],
            "subtitle": "EMBEDDED",
            "publishedAt": 1776624843260
          }
          /* … 最多 30 条，按 publishedAt desc … */
        ]
      }
      /* … 其他字幕组 … */
    ],
    "total": 4
  }
}
```

### 字段

- `groups` 列表按 `lastUpdatedAt` 倒序（最近发过的排前面）。
- `items`：该字幕组对该番剧最近发布的资源，**最多 30 条**，按 `publishedAt` desc。可直接渲染"番剧 × 字幕组"维度的资源面板，避免单独拉 RSS。
  - `magnet` 可能为 `null`（仅有种子文件、未生成磁力时）。
  - `episodeKey` 是稳定的集数键（如 `"01"`、`"SP01"`），便于客户端排序 / 聚合。
  - `language`：`("CHS" | "CHT" | "JP" | "EN")[]`。
  - `subtitle`：`"EMBEDDED" | "EXTERNAL" | "BUILT_IN" | "NONE"`（内嵌 / 外挂 / 内封 / 无）。
- `slug` 与 [`/api/subtitle-groups`](/docs/open-api/subtitle-groups) 完全一致，可直接喂给 `/rss/anime.xml?groupSlug=…`。
- 如果该 `bgmId` 未上架或被隐藏，返回 `404 ANIME_NOT_FOUND`。

### 典型用法 — 给追番器拼 RSS

```bash title="两步拼出 RSS"
# Step 1: 知道 bgmId，拉它的字幕组列表
curl -s 'https://site.anibt.net/api/anime/groups?bgmId=543360' \
  | jq -r '.data.groups[].slug'
# kitauji-sub
# kirara-fantasia
# pre-s
# haruhana

# Step 2: 拼 RSS（多个字幕组、CHS+1080p）
curl 'https://anibt.net/rss/anime.xml?bgmId=543360&groupSlug=kitauji-sub,kirara-fantasia,pre-s,haruhana&language=CHS&resolution=1080p&limit=50'
```

### 缓存与限流

| 项目             | 值                                                                              |
| ---------------- | ------------------------------------------------------------------------------- |
| 速率限制         | **30 次 / 60 秒 / IP**（与 `/api/seasons/anime` 共用窗口）                        |
| `Cache-Control`  | `max-age=300, s-maxage=300, stale-while-revalidate=3600`（**5 分钟新鲜 + 1h SWR**） |
| 条件请求         | `If-None-Match`（`W/"ag-{bgmId}-{count}-{lastMs}-{urlHash}"`）、`If-Modified-Since` |

<Callout type="info">
  当字幕组发新 release，`lastUpdatedAt` 跳动，ETag 失效，缓存自然刷新。轮询频率建议**不要短于 5 分钟**——更高频对你没有任何收益。
</Callout>
