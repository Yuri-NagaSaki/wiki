**AniBT** 是一个面向动漫的公共种子追踪器，专注服务动漫与字幕组。我们希望给字幕组一个可靠的发布平台，也让用户和工具更容易自动追到想看的内容。

这份 Wiki 按用途分两条主线：**字幕组发布资源** 和 **用户/开发者消费数据**。挑你属于的那一边读起。

---

## 我是字幕组，想发布资源

完整流程：申请入驻 → 配站点同步 → 制种用我们的 tracker → 用 API 发布。

  />
  <Card
    title="② 站点同步"

    description="把 AniBT 发布自动同步到 nyaa、萌番组、末日站、Mikan、ACG.rip。"
  />
  <Card
    title="③ Tracker 地址"
    href="/docs/tracker"

</Cards>
